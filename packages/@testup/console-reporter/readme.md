# TestUp Console Reporter

> Part of the [TestUp](https://github.com/testup) test running suite.

Basic TestUp reporter. Output mocha-alike reports to console.

## Install

```shell
npm i @testup/console-report
```

## Options

### `indent`

```
string = '  '
```

Indentation string.

### `successMark`
```
string = ''
```

Succeeded test marker.

### `failureMark`
```
string = ''
```

Failed test marker.
