# Testup Tap Reporter

> Part of the [TestUp](https://github.com/testup) test running suite.

Testup [TAP]() reporter.

## Install

```shell
npm i @testup/tap-reporter
```

## Options

## `lineLength`
```
number = 80
```

Specify maximum line length for line wrapping.
